# 🚗 Sistema de Gestão de Acesso de Estacionamento

Sistema de Gestão de Estacionamento feito em Java Swing, feito para uma empresa fictícia (Tech Solutions S.A.), aplicando conceitos práticos de banco de dados relacional com MySQL.

## 🔨 Tecnologias e Ferramentas Utilizadas

* Java SE 8
* Java Swing
* MySQL
* Netbeans IDE 8.0.2

### 💻 O que envolve o projeto?

O projeto possui um sistema de CRUD completo focado na gestão de um estacionamento, incluindo o cadastramento completo de visitantes do estacionamento e a conexão com o veículo cadastrado via CPF.

Além disso, também permite o registro de entrada e saída de veículos do estacionamento, com data e hora. Ademais, o registro possui verificação via STATUS, possibilitando o usuário saber se o veículo está estacionado ou se está fora.
